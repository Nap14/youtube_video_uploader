# Zoom Video Automation System 🚀

Інструмент для автоматизації повного циклу завантаження відео.

## 🏗️ Архітектура та Версії

Додаток представлений у двох форматах для різних сценаріїв використання:

### 1. Графічний інтерфейс (`main.py` / `GUI.exe`)
Повнофункціональний додаток із сучасним дизайном.
- 📊 **Dashboard**: Моніторинг статусу та ручний запуск завантажень.
- 📅 **Automation Planner**: Потужний планувальник для фонової роботи.
- ⚙️ **Settings**: Зручне налаштування токенів та шляхів без редагування JSON.
- 🔔 **System Tray**: Згортання в область сповіщень для непомітної роботи.

### 2. Термінальна версія (`cli.py` / `CLI.exe`)
Легка консольна утиліта для швидкої роботи.
- ⚡ **Швидкість**: Миттєвий запуск без вікон.
- ⌨️ **Інтерактивність**: Зручне меню вибору режимів přímo в консолі.
- 🛠️ **Автоматизація**: Ідеально підходить для запуску через сторонні скрипти.

## ⚙️ Налаштування

### 🔑 YouTube API
Покладіть файл `client_secret.json` у папку `configs/`. При першому запуску відкриється браузер для авторизації.

### 📅 Розклад та ID курсів (`configs/schedule.json`)
Вкажіть дані ваших курсів:
```json
{
  "Monday": {
    "16:10": {
      "name": "Назва курсу",
      "thumbnail": "thumbnail.png",
      "lms_course_id": "123456"
    }
  }
}
```

## 🚀 Запуск

### Як почати роботу:
1. Використовуйте **`ZoomAutomation_GUI.exe`** для налаштування системи та розкладу.
2. Увімкніть **Enable Automation Service** на вкладці Planner.
3. Програма може бути згорнута в трей і працюватиме повністю автоматично.
4. Якщо потрібно терміново завантажити файл вручну — скористайтеся **`ZoomAutomation_CLI.exe`**.
