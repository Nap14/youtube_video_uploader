# Zoom -> YouTube Uploader 🚀

Автоматизація завантаження записів занять із Zoom на YouTube з організацією по плейлистах та генерацією звітів.

## Особливості
- 📂 **Автоматичний пошук**: Знаходить `.mp4` файли у папках Zoom.
- 📅 **Розумний розклад**: Визначає назву курсу на основі дати та часу створення запису.
- 📺 **YouTube API**: Автоматичне завантаження як "Unlisted" (Не для всіх).
- 🎼 **Плейлисти**: Автоматично створює або знаходить плейлист для конкретної групи та додає туди відео.
- 🖼️ **Обкладинки**: Можливість встановлення кастомних thumbnails для кожного курсу.
- 🗑️ **Безпечне видалення**: Після успішного завантаження папка Zoom переміщується в Кошик.

## Налаштування

### 1. Google API
Покладіть файл `client_secret.json` (отриманий у Google Cloud Console) у корінь папки проекту.

### 2. Розклад (`schedule.json`)
Створіть файл `schedule.json` у корені проекту за таким зразком:
```json
{
  "Monday": {
    "15:00": {
      "name": "Python Група 1",
      "thumbnail": "python.jpg"
    },
    "19:00": "Unity Пн+Ср 19:00"
  },
  "Wednesday": {
    "19:00": "Unity Пн+Ср 19:00"
  }
}
```
*Нотатка: Назви днів тижня мають бути англійською (`Monday`, `Tuesday`, і т.д.).*

### 3. Обкладинки
Всі картинки (наприклад, `python.jpg`) мають лежати у папці `thumbnails/`.

## Запуск

### Windows
Запустіть файл **`Запуск.bat`**. При першому запуску він сам створить віртуальне середовище та встановити залежності.

### macOS / Linux
Використовуйте термінал:
```bash
chmod +x run.sh
./run.sh
```
