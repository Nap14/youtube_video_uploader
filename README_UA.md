# Zoom -> YouTube -> LMS Sync 🚀

Автоматизація завантаження записів занять із Zoom на YouTube з подальшою синхронізацією посилань у систему навчання SendPulse LMS.

## Особливості
- 📂 **Автоматичний пошук**: Знаходить `.mp4` файли у папках Zoom.
- 📅 **Розумний розклад**: Визначає назву курсу та зіставляє його з ID у SendPulse.
- 📺 **YouTube API**: Автоматичне завантаження як "Unlisted" (Не для всіх) та додавання у плейлисти.
- 🎓 **LMS Синхронізація (SendPulse)**: Автоматично знаходить потрібний урок за датою та додає жовту кнопку "Відео" з посиланням на запис.
- 📊 **Гнучкі звіти**: Створює зручні `.txt` звіти для людей та `.csv` для автоматизації.
- ⚙️ **Три режими роботи**:
  1. **Full Cycle**: YouTube + LMS (Повний цикл).
  2. **YouTube Only**: Тільки завантаження на відеохостинг.
  3. **LMS Sync Only**: Тільки оновлення посилань у школі на основі готового звіту.

## Налаштування

### 1. YouTube API
Покладіть файл `client_secret.json` у папку `configs/`.

### 2. LMS та Глобальні налаштування (`configs/config.json`)
Створіть файл `config.json` у папці `configs/` для збереження токенів та налаштувань:
```json
{
  "LMS_TOKEN": "ваш_bearer_token",
  "SCHOOL_ID": "18828",
  "REPORTS_DIR": "reports"
}
```
*Токен можна отримати в Network інструментах браузера при вході в кабінет SendPulse.*

### 3. Розклад та ID курсів (`configs/schedule.json`)
Тепер у розкладі потрібно вказати `lms_course_id`. Файл має лежати в `configs/`:
```json
{
  "Monday": {
    "16:10": {
      "name": "Назва курсу",
      "thumbnail": "image.png",
      "lms_course_id": "12345"
    }
  }
}
```

## Запуск

### Windows
Запустіть файл **`Run.bat`**. Оберіть потрібний режим у консолі.

### Інтерфейс
Програма використовує вікна вибору папок та файлів, тому вам не потрібно вручну копіювати довгі шляхи.
