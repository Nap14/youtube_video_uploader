# Інструкція з налаштування та запуску Zoom Uploader 🚀

Цей інструмент автоматизує завантаження записів Zoom на YouTube, розкладає їх по плейлистах та створює звіти.

---

## 🏗️ 1. Підготовка Google API

Щоб програма могла завантажувати відео на ваш YouTube-канал, потрібно отримати файл з ключами:

1. Перейдіть у [Google Cloud Console](https://console.cloud.google.com/).
2. Створіть новий проект (або виберіть існуючий).
3. У розділі **"APIs & Services" > "Library"** знайдіть та ввімкніть **"YouTube Data API v3"**.
4. У розділі **"OAuth consent screen"** налаштуйте екран згоди (виберіть тип "External" та додайте свій email у "Test users").
5. У розділі **"Credentials"** натисніть **"Create Credentials" > "OAuth client ID"**.
6. Оберіть тип **"Desktop app"**.
7. Завантажте отриманий JSON-файл і перейменуйте його на **`client_secret.json`**.
8. Покладіть цей файл у папку з програмою.

---

## ⚙️ 2. Створення необхідних файлів

Перед запуском переконайтеся, що у папці з програмою є такі файли:

### 📅 Розклад (`schedule.json`)
Створіть цей файл, щоб програма знала, до якої групи відноситься запис.
```json
{
  "Monday": {
    "15:00": { "name": "Python Група 1", "thumbnail": "python.jpg" },
    "19:00": "Unity Пн+Ср 19:00"
  },
  "Wednesday": {
    "19:00": "Unity Пн+Ср 19:00"
  }
}
```
*Назви днів тижня: `Monday`, `Tuesday`, `Wednesday`, `Thursday`, `Friday`, `Saturday`, `Sunday`.*

### 🖼️ Обкладинки (`thumbnails/`)
Якщо ви вказали `thumbnail` у розкладі, створіть папку `thumbnails` та покладіть туди відповідні зображення (наприклад, `python.jpg`).

---

## 🚀 3. Варіанти запуску

### Варіант А: Запуск із початкового коду (GitHub)
Це найкращий варіант для розробників або якщо ви хочете бачити детальні логи.

1. Переконайтеся, що встановлено **Python 3.10+**.
2. Завантажте код із репозиторію.
3. Перейдіть у папку проекту через термінал/командний рядок.
4. **Для Windows:** Просто запустіть файл `Run.bat`. Він сам створить віртуальне середовище (`venv`) та встановить бібліотеки.
5. **Для macOS / Linux:**
   ```bash
   chmod +x run.sh
   ./run.sh
   ```

### Варіант Б: Запуск готового EXE-файлу
Якщо у вас є зібраний файл `ZoomUploader.exe`:

1. Створіть нову папку і покладіть туди:
   - `ZoomUploader.exe`
   - `client_secret.json` (ваш ключ API)
   - `schedule.json` (ваш розклад)
   - Папку `thumbnails` (якщо використовуєте обкладинки)
2. Просто запустіть `ZoomUploader.exe`.

---

## 💡 Важливі примітки
*   **Перший запуск:** Програма відкриє браузер і попросить вас увійти в Google-аккаунт. Після авторизації з'явиться файл `token.json` — не видаляйте його, він дозволяє програмі працювати без повторного входу.
*   **Вибір папки Zoom:** При першому запуску (або якщо шлях не вказано) програма попросить вас обрати папку, де лежать записи Zoom, через зручне вікно вибору.
*   **Безпека:** Після успішного завантаження програма автоматично переносить папку із записом у **Кошик**.
