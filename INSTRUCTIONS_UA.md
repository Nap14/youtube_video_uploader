# Інструкція з налаштування Zoom-YouTube-LMS Uploader 🚀

Цей інструмент автоматизує весь шлях: від пошуку запису Zoom до появи кнопки "Відео" у вашому курсі на SendPulse.

---

## 🏗️ 1. YouTube API (client_secret.json)
1. Перейдіть у [Google Cloud Console](https://console.cloud.google.com/).
2. Створіть новий проект.
3. Ввімкніть **"YouTube Data API v3"**.
4. Створіть **OAuth client ID** (тип "Desktop app").
5. Завантажте JSON-файл і перейменуйте його на **`client_secret.json`** у папку проекту.

---

## 🎓 2. SendPulse LMS (config.json)
Для синхронізації з навчальною платформою програмі потрібен токен доступу.

1. У папці з програмою створіть файл **`config.json`**:
```json
{
  "LMS_TOKEN": "ВСТАВИТИ_ТОКЕН_ТУТ",
  "SCHOOL_ID": "18828",
  "REPORTS_DIR": "reports"
}
```

### Як отримати токен:
1. Зайдіть у кабінет SendPulse у браузері Chrome.
2. Натисніть **F12** (або ПКМ -> Оглянути код) та перейдіть на вкладку **Network**.
3. Оновіть сторінку.
4. Знайдіть будь-який запит, що починається на `api/v1/courses` або `get-user-info`.
5. У розділі **Request Headers** знайдіть рядок `Authorization: Bearer <багато_символів>`.
6. Скопіюйте частину після слова `Bearer` — це і є ваш токен.

*Примітка: Токен діє обмежений час. Якщо скрипт видає помилку 401 — оновіть токен у `config.json`.*

---

## 📅 3. Розклад та ID курсів (schedule.json)
Програма має знати, до якого курсу в SendPulse відноситься заняття.
ID курсу можна знайти в адресному рядку браузера, коли ви редагуєте курс (`.../edu/courses/<ID>/...`).

```json
{
  "Monday": {
    "19:00": {
      "name": "Unity Pro",
      "lms_course_id": "35014"
    }
  }
}
```

---

## 🚀 4. Вибір режиму роботи
При запуску `Run.bat` з'явиться меню:

1. **Full Cycle**:
   - Шукає нові записи Zoom.
   - Завантажує на YouTube.
   - Переносить папку в Кошик.
   - Одразу додає посилання в SendPulse.
2. **YouTube Only**:
   - Тільки завантаження та Кошик. Створює CSV-звіт, який можна використати пізніше.
3. **LMS Sync Only**:
   - Відкриє провідник для вибору CSV-звіту.
   - Синхронізує відео з курсами без повторного завантаження файлів.

---

## 📊 5. Звіти
Всі результати зберігаються у папці `reports/`:
- **Youtube_Report_...txt**: Зручний список посилань для перегляду.
- **youtube_report_...csv**: Технічний файл для завантаження в LMS пізніше.
- **lms_report_...csv**: Статус оновлення уроків на платформі.
